## Reflection

### What did you learn this week?
This week, I learned how to use Git and GitHub for version control in Python projects. I also practiced writing functions that follow the Single Responsibility Principle and used refactoring techniques to improve existing code.

### What did you find challenging?
It was a bit challenging to properly structure the functions without relying on global variables. Also, ensuring meaningful commit messages and using GitHub correctly required careful attention.

### How could you improve your learning?
I can improve by practicing more with Git and GitHub using both the command line and IDE. Also, I will continue to refactor code regularly to write cleaner and more modular programs.

### What resources did you use or find helpful?
The GitHub guides, lecture materials, and PyCharm's built-in Git tools were all helpful. Refactoring tools in PyCharm made it easier to extract functions properly.
